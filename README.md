FlaskOfMemes is a simple web application built with **Flask** that fetches and displays Pepe memes from the [OnlyPepes API](https://api.onlypepes.xyz/). Perfect for a quick laugh or exploring random memes with a single click!  

---

## Features

- Fetches **random Pepe memes** dynamically from an API  
- **Error handling** for network issues or failed API responses  
- Displays memes in a clean, simple web interface  
- Uses Flask templates for **dynamic HTML rendering**
